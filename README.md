# Pong in Assembly
### Description:
Pong in assembly is a take on the classic pong game. This project was built using NASM assembly for game logic and XLib for rendering.

### Installation:
- Clone and cd into repo
- run `cmake . -B build`
- run `cmake --build build`

### Run
- run `build/AssemblyGame`
